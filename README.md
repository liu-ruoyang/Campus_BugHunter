# Campus BugHunter

## 📱 Project Overview
A Flutter + Firebase mobile application for campus-based peer debugging.

---

## 🚀 Features
- User authentication (login/register)
- Bounty-based request system
- Helper & requester roles
- Wallet system (mock)

---

## 🛠️ Tech Stack
- Flutter (Frontend)
- Firebase (Backend)
- GitHub (Version Control)
